## О проекте

Современное веб-приложение на Nuxt 3 для ведения истории мероприятий, наград, компетенций и проектов. Тёмный фиолетовый дизайн с адаптивной вёрсткой под все устройства.

| Технология | Версия | Назначение |
|---|---|---|
| Nuxt 3 | ^3.17.4 | SSR / файловый роутинг / полный стек |
| Vue 3 | ^3.5.16 | UI + Composition API |
| Vue Router | ^4.5.1 | Клиентская навигация |
| TypeScript | — | Типизация (tsconfig) |
| SASS | ^1.89.1 | Стили, дизайн-токены через переменные |

---

## Возможности

- Дашборд профиля: карточка пользователя с аватаром, уровнем, прогресс-баром и балансом «окто-коинов»
- Достижения: витрина полученных наград с возможностью прокрутки
- Текущее событие: блок с описанием активного мероприятия и кнопкой «Подробнее»
- История событий: лента прошедших мероприятий
- История достижений: список всех полученных наград
- Проекты: карточки с описанием участвуемых проектов
- Компетенции: визуальный прогресс освоения навыков по направлениям с кнопкой «Начать обучение»
- Новостная лента: боковая панель с актуальными новостями
- Навигационный сайдбар: иконки быстрого доступа
- Мобильная навигация: нижний таб-бар автоматически появляется на экранах уже 600px

## Быстрый старт

Нужен [Node.js](https://nodejs.org/) версии 18 или новее.

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Topchikdan/Events-and-achievements.git
cd Events-and-achievements

# 2. Установите зависимости
npm install

# 3. Запустите в режиме разработки
npm run dev
```

Откройте адрес из терминала (обычно http://localhost:3000).

## Структура проекта

```
.
├── app.vue                         # корневой компонент (NuxtLayout / NuxtPage)
├── nuxt.config.ts                  # конфигурация Nuxt
├── pages/
│   ├── index.vue                   # главная: профиль, проекты, компетенции, новости
│   └── dashboard.vue               # дашборд: достижения, история событий и наград
├── components/
│   ├── AppSidebar.vue              # вертикальный сайдбар с иконками навигации
│   ├── SidebarIcons.vue            # SVG-иконки для сайдбара
│   ├── BottomNavMobile.vue         # нижняя навигация для мобильных (<600px)
│   ├── ProfileCard.vue             # карточка профиля, уровень, достижения, текущее событие
│   ├── ProjectCard.vue             # карточки проектов
│   ├── CompetenciesCard.vue        # блок компетенций с прогресс-индикатором
│   ├── NewsSidebar.vue             # боковая панель новостей
│   ├── AchievementsHeader.vue      # заголовок страницы достижений
│   ├── AchievementHistoryList.vue  # список истории достижений
│   └── EventHistoryList.vue        # список истории событий
├── assets/
│   ├── achievement.jpg
│   └── styles/
│       ├── variables.scss          # дизайн-токены (цвета, тени, радиусы)
│       └── main.scss               # глобальные стили
└── server/
    └── tsconfig.json               # конфигурация TypeScript для сервера
```

## Навигация

| Путь | Описание |
|---|---|
| `/` | Главная — профиль, проекты, компетенции |
| `/dashboard` | Дашборд — история событий и достижений |
| `/events` | Список событий |
| `/favorites` | Избранное |
| `/settings` | Настройки |
| `/cart` | Корзина |
| `/backpack` | Рюкзак |
| `/help` | Помощь |

---

## Адаптивность

| Брейкпоинт | Поведение |
|---|---|
| > 1200px | Трёхколоночный грид: профиль / центр / новости |
| ≤ 1200px | Колонки сжимаются пропорционально |
| ≤ 900px | Одноколоночный layout |
| ≤ 600px | Сайдбар скрыт, появляется нижняя мобильная навигация |

---

## Дизайн-токены

Все базовые значения вынесены в `assets/styles/variables.scss`:

```scss
$bg-main:       #191942;   // основной фон (тёмно-синий)
$bg-secondary:  #23235a;   // фон сайдбара
$text-main:     #fff;
$text-secondary:#bdbdd1;
$text-accent:   #be73fa;   // акцентный фиолетовый
$card-radius:   20px;
$box-shadow:    0 6px 32px 0 rgba(67, 62, 129, 0.18);
```
